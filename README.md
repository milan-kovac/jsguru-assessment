# 🤖 JsGuru Assessment 💻

## 🧰 Requirements

NodeJS (v14)
npm (package manager)

## Installation

```bash
⚙️ npm install
```

## Development Setup

1. Create `.env` file. (You can copy `.env.example` file)
2. Run `npm install` to install all the dependencies.
3. Run `npm start` or `npm start:dev` to start the server.

## Running the app

```bash
Development
👷 npm start

Watch mode
🧑‍🏭 npm start:dev

Production mode
🚀 npm start:prod
```
